# notes-node

> Nodejs note app.

Install the dependencies.

```sh
$ npm install
```

To add a new note.

```sh
$ node app.js add --title="Note title" --body="Note body"
```

To get all notes.

```sh
$ node app.js list
```

To read a note.

```sh
$ node app.js read --title="Note title"
```

To remove a note.

```sh
$ node app.js remove --title="Note title"
```
